Additional Custom Parameters
createToolbar Creates the <%=$LICPROD%>'s toolbar.
toolbarVisible Starts connection with the toolbar visible.
permissions.expandToolbar Allows start connection with the toolbar expanded.
permissions.showFileTransfer Enables/disables file transfer option.
disableDisconnect Enables/disables disconnect option.
checkBeforeWindowClose When false, it skips the user confirmation popup of the onBeforeUnload event.
noSsnDialog Avoids the share session popup dialog.
noShowConnectionClosePopup When true, it skips the session closed message popup.
noShowConnectionDisconnectPopup When set to true, the 'Connection Disconnect' message popup will be skipped.
